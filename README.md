# Junior Cycle Mathematics

PreTeXt textbook for Junior Cycle Mathematics aligned with the Irish specification.

## Project Structure

- `source/` - PreTeXt source files organized by curriculum strand
- `assets/` - Images and other static assets
- `generated-assets/` - Auto-generated images
- `publication/` - Publication configuration files
- `output/` - Built HTML and PDF output

## Building the Book

### Build HTML
```bash
python -m pretext build web
```

### Build PDF
```bash
python -m pretext build print
```

### View HTML locally
```bash
python -m pretext view web
```

## Curriculum Strands

1. **Unifying** - Mathematical thinking, reasoning, communication, problem-solving
2. **Number** - Number systems, operations, fractions, decimals, percentages, ratio
3. **Geometry and Trigonometry** - Shapes, constructions, coordinates, transformations
4. **Algebra** - Expressions, equations, patterns, sequences
5. **Functions** - Relationships, graphs, rates of change
